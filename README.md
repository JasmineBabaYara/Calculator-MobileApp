# Calculator-MobileApp

- Frontend - ReactNative, JavaScript

## Required Software

- Node.js
- Expo CLI

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/JasmineBabaYara/Calculator-MobileApp.git
~~~

Install dependencies

~~~bash  
expo install
~~~

Start the application

~~~bash  
expo start
~~~  

Open on android, ios or your browser
